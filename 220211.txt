1. 통계
- 기술통계학(descriptive statistics): 데이터를 정리 요약 설명하는 방법론
- 추론통계학(inferential statistics): 표본 데이터를 분석해 모집단의 여러 특성을 추측하는 방법론

2. 통계의 역사
1) 통계 
- 가설 -> 검증
- 기술 통계 분석
- 추론 통계 분석

2) 데이터 마이닝
- 군집, 연관분석
- 예측 모델링
- 텍스트 마이닝

3) 빅데이터
- 머신러닝
- 딥러닝

4) AI

3. 데이터의 이해
1) 데이터와 그래프
- 변수: 통계학에서는 조사 목적에 다라 관측된 자료값을 의미하며, 관측 값은 자료(Data)가 됨
- 질적 자료: 범주로 구분하여 표현할 수 있는 데이터로 명목형, 순서형 데이터로 구분 (ex. 남자1, 여자2)
- 양적 자료: 숫자의 크기가 의미를 갖는 데이터로 이산형 데이터, 연속형 데이터로 구분
- 데이터를 분석하는 방법(EDA)
    - 데이터 분석 프로젝트 초기에 가설 수립을 위해 사용
    - 적절한 모델 및 기법의 선정
    - 변수간 트렌드, 패턴, 관계 등을 찾고 통계적 추론을 기반으로 가정을 평가
    - 이상치 발견
- 데이터 시각화: 도표라는 수단을 통해 정보를 명확하고 효과적으로 전달
    - 시간 시각화: 막대, 누적막대, 점
    - 분포 시각화: 파이, 도넛아트, 트리맵,  누적연속 그래프
    - 관계 시각화: 스캐터 플록, 버블차트, 히스토그램
    - 비교 시각화: 히트맵, 스타 차트, 평행 좌표계, 다차원 척도법
    - 공간 시각화: 지도 맵핑

2) 데이터의 기초 통계량 
- 기초통계량(=기술 통계량)
- 중심 경향치: 표본의 중심을 설명하는 값을 대표값 또는 중심경향치라고 함
    - 대표적으로 평균, 중앙값, 최빈값, 절사 평균 등이 있음
    - 평균()
    - 중앙값: 자료가 홀수일 경우 중앙값, 짝수일 경우 가운데 두개의 값을 산술 평균한 값
    - 최빈값(mode): 가장 많이 관측되는 값
- 산포도: 데이터가 흩어진 정도로, 중심경향치와 함께 고려
    - 범위, 사분위수, 분산, 표준편차, 변동 계수 등
    - 범위: 최대값-최소값
    - 사분위수: 데이터를 오름차순으로 정렬하여 4등분하고 첫 번째를 제1사분위수(Q1)~Q4 라고 함
    - 백분위수: 데이터를 오름차순으로 정렬하여 주어진 비율에 의해 등분한 값 
    - ***분산(σ²):데이터의 분포가 얼마나 흩어져 있는지 알 수 있는 측도
        - 각각의 값들의 편차 제곱합으로 계산 
    - 표준편차(s)
    - 변동 계수(CV, Coefficient of Variation): 표준편차를 평균으로 나눠서 두 분포를 비교 
- 분포: 정규분포모양
- 왜도: 분포가 얼마나 비대칭적인지 표현하는 지표
    - 0이면 좌우 대칭
    - 0보다 클수록 우측 꼬리가 길고, 0보다 작을수록 좌측 꼬리가 김
- 첨도: 확률분포의 꼬리가 두꺼운 정도
    - 3에 가까우면 산포도가 정규분포에 가까움 
    - 3보다 작을 경우 꼬리가 얇은 분포(아래로 넓적) 
    - 3보다 클 경우 꼬리가 두꺼운 분포(위로 뾰족)


4. 확률
1) 확률: 모든 경우의 수에 대한 특정 사건이 발생하는 비율
    - 고전적 정의: 일어날 수 있는 경우의 수:가능한 모든 경우의 수

2) 표본 공간(S): 어떤 실험에서 나올 수 있는 모든 가능한 결과들의 집합
- 사건 A가 일어날 확률 P(A), 표본공간(S)가 유한집합, 표본 공간의 모든 원소들이 일어날 확률이 같으면
    (ex. 주사위, 동전) P(A) = 사건A가 일어날 원소의 수 / 표본공간S의 원소의 수

3) 통계적 확률 정의: 어떤 시행을 N번 반복했을 때, 사건 A에 해당하는 결과가 r번 일어난 경우 = r/N 이고, 이는 사건A가 일어날 상대도수
- N이 무한히 커지면 상대도수는 일정한 수로 수렴 -> 이 극한값을 사건A의 통계적 확률 또는 경험적 확률이라고 함

4) 확률의 성질
- 합사건: 사건A 또는 사건B가 일어날 확률 (A합B)
- 곱사건: 사건A와 사건B가 동시에 일어날 확률 (A교B)
- 배반사건: 사건A와 사건B가 동시에 일어날 수 없는 경우 (A교B=공집합)
- 여사건: 사건A가 일어나지 않을 확률 (A여)
- 확률의 덧셈법칙, 배반사건일때 교집합=0, a+a여=1

5) 순열: 순서를 고려하여 n개 중 r개를 뽑아서 배열하는 경우의 수 = n!/(n-r)!
6) 조합: 순서를 고려하지 않고 n개 중 r개를 뽑아서 배열하는 경우의 수 = nPr/r! = n!/r!(n-r)!
